# Skill Foundry bundle

This bundle contains a cross-platform meta-skill for building and organizing skills.

## Included
- `agents/skills/skill-foundry/SKILL.md`
- `agents/catalog.yaml` (meta-skill entry)
- `agents/catalog-product-management.yaml` (taxonomy and overlaps for PM skills in `.agents/skills/<name>/`; routing text stays in each `SKILL.md`)
- `agents/taxonomy.md`
- references for lifecycle, evals, descriptions, and anti-patterns
- assets for scaffolding new skills and catalog entries

## Intended use
Use this bundle as a starting point for a shared Agent Skills library that can work across Claude, Codex, ChatGPT-compatible tools, and other environments that support the open Agent Skills format.
